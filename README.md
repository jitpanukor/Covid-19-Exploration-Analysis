# Coronavirus (COVID-19) World Exploration Analysis

In this project, we analyzed the COVID-19 death rates and vaccination progress across the world to uncover key trends and insights. The goal was to better understand how vaccinations impacted the number of COVID-19 deaths globally and identify patterns in different regions.
